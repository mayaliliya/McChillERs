POWERED ARM

Our Goal

Powered Arm uses 3D printing technology to build personalized prosthetic designs for a robotic arm. Currently the team has finished designing the arm and is wokring on controling servo motors using EMG signals to pull the fingers closed. The team hopes to use a machine learning algorithm to employ a more efficient signal processing algorithm.
